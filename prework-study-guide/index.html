HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An HTML document includes the content that users see when they look at the webpage in a browser.
  • HTML stands for Hypertest Markup Language.
  • HTML is one of the three cornerstone languages used in web developement.
  • HTML is used to build a sturdy frame for a website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS describes how HTML elements should be displayed.
  • CSS stands for Cascading Style Sheets (CSS)
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Besides text, we can also apply CSS rules to images.
  • CSS can also make a website simpler to use with a polished design and consistent organization.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Git commands can be used to go back to earlier versions of our if we need to.
  • Git is the free and open source distributed version control system that's responsible for everything GitHub related that happens locally on your computer.
  • git commit -m "[descriptive message]": will commit your staged content as a new commit snapchot.
  • git branch: Lists your branches an * will appear next to the currently active branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are some of the fundamental building blocks of JavaScript.
  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive.
  • The .js file extension indicates that this file is a JavaScript file.
  • A function is different from a for loop or conditional statement because functions do not automatically execute when the JavaScript file is run.
  • A function's name is followed by parentheses ().
